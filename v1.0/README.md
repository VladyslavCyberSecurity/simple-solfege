# Simple Solfege

Simple Solfege — це мінімалістичний Android-застосунок для тренування слуху та сольфеджіо.  
Перший функціонал — інтерактивне піаніно з нотами C–B, які відтворюються через синусоїдальний генератор частоти (ToneGenerator).

## Функціонал (MVP)
- Піаніно з 7 клавішами: C D E F G A B
- Генерація синусоїдального тону для кожної ноти
- Підсвітка активної клавіші
- Чистий мінімалістичний інтерфейс на Jetpack Compose

## Технічні деталі
- Android Studio Hedgehog
- Kotlin + Jetpack Compose
- AudioTrack API для генерації звучання
- Чиста архітектура:
  - `audio/` — генерація тону
  - `logic/` — нотні дані
  - `ui/` — компоненти інтерфейсу
  - `MainActivity` — точка входу

- **main** — стабільна робоча версія застосунку.
